# CAR RENTAL PROJECT
<!-- -----------------REQURIEMENTS--------------------------- -->
1. Web Server (Apache)
2. PHP (at-least version 5.6)
3. Database -> My SQL
4. Any Web browser like Chrome
<!-- -----------------INSTALLATION--------------------------- -->
1. INSTALL XAMPP "https://www.apachefriends.org/index.html"
2. START APACHE AND MYSQL
3. CLONE https://github.com/AmeliaHe98/304_project.git IN XAMPP/HTDOCS
4. GO TO http://localhost:<i>port number</i>/phpmyadmin
5. Create new database named <b>carrental</b>
6. Insert carrental.sql from cloned project to <b>carrental</b> database from phpmyadmin
7. Visit http://localhost:8080/304_project/
